# MyAndroidApp

A modern Android application built with **Kotlin**, **Jetpack Compose**, and **Clean Architecture**.

## 🎯 Features
- 🎨 Material Design 3 UI with Jetpack Compose
- 🏗️ Clean Architecture with modular structure
- 💉 Dependency Injection with Hilt
- ⚡ Built with Kotlin Coroutines
- ✅ Comprehensive testing

## 📋 Requirements
- Android API 24+
- JDK 11+
- Gradle 8.2+
